# maroonlis-utils

## Usage

- install

```shell
npm install maroonlis-utils --save

# or

yarn add maroonlis-utils
```

- example

```js
import { isNotEmptyArray, isHTMLString } from 'maroonlis-utils'

isNotEmptyArray([])
isNotEmptyArray([123])
isHTMLString('')
isHTMLString('<div>test</div>')
```
